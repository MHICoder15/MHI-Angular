RxJS (Reactive Extensions for JavaScript) is a JavaScript library, that allow us to work with the asynchronous data stream with the help of observables.

An Observable is like a pipeline through which data flows.
It can emit multiple values over time,
and you can subscribe to an Observable to receive these emitted values.

An observable is useless until an observer subscribes to it.
An observer is an object which consumes the data emitted by the observable.

An observer keeps receiving data values from the observable until the observable is completed,
or the observer unsubscribes from the observable.

Observables Promises
Emit multiple values over the period of time, Also called streaming of data . Emit a single value at a time.
Are lazy: they're not executed until we subscribe to them using the subscribe() method. Are not lazy: execute immediately after creation.
Have subscriptions that are cancellable using the unsubscribe() method. Are not cancellable.
